# Chess Scan Web

A web application for extracting chess data from images using [chess-scan](https://www.npmjs.com/package/chess-scan).

Upload a chessboard image and instantly:

- Generate FEN notation
- View prediction confidence
- Extract the detected board image
- Extract all 64 individual square images
- Edit and refine positions using an interactive board editor

Chess Scan Web is powered by my open-source Chess Scan project:

- [Source Code](https://github.com/sathwikv2005/Chess-Scan)
- [npm Package](https://www.npmjs.com/package/chess-scan)
